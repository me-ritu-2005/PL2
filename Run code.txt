🔹 Procedure to Run the Program (Row Count)
Open any Python IDE (VS Code).
Create a new Python file and save it as
count_rows.py
Create a CSV file named
data.csv
Write some sample data in the CSV file.
Place both files in the same folder.
Open Terminal / Command Prompt.
Navigate to the folder where the Python file is saved.
Run the program using the command:

python count_rows.py
The program will read the CSV file and count the total rows.

Procedure to Run the Program (JSON to CSV Conversion)
Open Python IDE such as VS Code.
Create a Python file and save it as
json_to_csv.py
Create a JSON file named
data.json
Write the sample JSON data in the file.
Place both files in the same folder.
Open Command Prompt / Terminal.
Navigate to the folder where the program is saved.
Run the program using the command:

python json_to_csv.py
After execution, a new file named data.csv will be created automatically.
